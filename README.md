# JSON PLACEHOLDER GRAPHQL

A port of the JSON placeholder API to GraphQL

## Application

### Requirements

- Node >= 16
- NPM

### How to run

1. Install the dependencies

```sh
$ npm i
```

2. Run the `fetch` script

```sh
$ npm run fetch
```

3. Start the app

- Development mode: `npm run dev`
- Production mode: `npm run start`
